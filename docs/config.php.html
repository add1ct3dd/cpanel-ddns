config.php is a file that defines the following constants:

            define('CPANEL_DOMAIN', <the domain your cpanel server is located at>);
            define('CPANEL_UN', <your cpanel username>);
            define('CPANEL_PW', <your cpanel password>);
            define('ZONE_DOMAIN', <the domain of the dns zone you want to edit>);

            /*
            * What type of ACL are we using?
            * 
            * single = a single IP address
            * multi = an array of single ip addresses
            * subnet = an array of host/mask subnet pairs.
            * 
            define('IP_ACCESS_MODE', <single, multi, subnet>);

            define('ALLOWED_IPS', <either a single ip or an array of ips that are 
            allowed to access the dns records>);

            It will need to be created and placed in the same directory as index.php
